# Microsoft News Aggregator

AI-powered news aggregator for Microsoft/Power Platform content.

## Features
- Auto-collects news from Microsoft sources
- AI summarizes articles
- Posts to Planka kanban for content ideas
- Daily/weekly digest generation

## Stack
- Python 3.12
- RSS feed parsing
- OpenAI API
- Planka integration
- GitHub Actions scheduling
